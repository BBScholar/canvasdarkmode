FAQ


Q: How do I report an issue I am having with the extension?

A: Click the extension's icon in your toolbar, and scroll down to "Report problem via". Then select a medium to report your issue.


Q: How do I install the beta version of the extension?

A: The beta version only supports Chromium browsers. You will have to download the source code from Github and sideload it yourself for now


Q: Does the extension work on other browsers besides Chrome?

A: Dark Mode for Canvas is developed and tested on Google Chrome, but it should work with browsers such as Brave and Yandex. Firefox support is coming soon
